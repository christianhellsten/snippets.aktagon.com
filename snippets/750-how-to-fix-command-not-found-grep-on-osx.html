How to fix "command not found:  grep" on OSX

This issue will bite everyone who touches a keyboard. This fixes, e.g., the following issues: \* command not found:  grep (Shell) \* undefined local variable or method (Ruby) \* First, do this: ```bash mkdir -p ~/Library/KeyBindings/ vim ~/Library/KeyBindings/DefaultKeyBinding.dict ``` Then paste this: ```bash { "~ " = ("insertText:", " "); } ``` Save the file. Quit and reopen applications to apply the new key bindings in DefaultKeyBinding.dict.
