Inspecting a live Ruby process

Inspecting a live Ruby process: On Linux: ```bash strace -p ``` On Mac: ```bash dtruss -p ``` ```bash gdb -p (gdb) t a a bt (gdb) call (void)rb_backtrace() ``` See https://blog.newrelic.com/engineering/debugging-stuck-ruby-processes-what-to-do-before-you-kill-9/
