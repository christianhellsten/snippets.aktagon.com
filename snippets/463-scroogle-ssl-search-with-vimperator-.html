Scroogle SSL search with vimperator.

A hack for using scroogle's SSL search as default search engine for :o, :p and :t commands of Vimperator. ```bash :bmark https://ssl.scroogle.org/cgi-bin/nbbwssl.cgi?Gw=%s -title="Scroogle SSL search" -keyword=scroogle :set defsearch=scroogle ```
