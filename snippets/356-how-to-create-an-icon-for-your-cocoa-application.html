How to create an icon for your Cocoa application

1. Create a 512x512 pixel icon with Photoshop and save it as a PSD file 2. Open the Icon Composer app and drag the PSD file into the first slot 3. Choose Save as from the menu 4. Import the file into XCode 5. Open Info.plist and set the Icon File setting to the file you just imported 6. Clean and recompile ``` ```
